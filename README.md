# unsignal

Composable signal bindings for frontend frameworks, built on top of [@preact/signals-core](https://github.com/preactjs/signals).

## Packages

| Package                               | Description                 |
| :------------------------------------ | :-------------------------- |
| [`@unsignal/react`](./packages/react) | Signal binding for React 19 |
| [`@unsignal/vue`](./packages/vue)     | Signal binding for Vue 3    |

## Getting Started

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run tests
pnpm test

# Type check
pnpm typecheck
```

## Requirements

- Node.js > 22.0.0
- pnpm 11+
